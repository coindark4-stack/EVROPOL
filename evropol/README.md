# Деплой на Vercel

Этот репозиторий — статический сайт (single `index.html` и папка `static/`). Можно быстро задеплоить на Vercel несколькими способами.

## Веб-интерфейс (рекомендуется)
1. Создайте новый репозиторий на GitHub/GitLab/Bitbucket и запушьте файлы.
2. Перейдите на https://vercel.com и подключите репозиторий (Import Project).
3. Выберите проект и нажмите Deploy — Vercel автоматически развернёт статический сайт.

## Через Vercel CLI
Установите CLI (если нужно):

```bash
npm i -g vercel
```

Зайдите в папку проекта и выполните:

```bash
vercel login
vercel
```

Следуйте подсказкам. По умолчанию Vercel определит, что это статический сайт и развернёт `index.html`.

## Особенности конфигурации
- В корне добавлен `vercel.json` с маршрутизацией и заголовками кеширования для статических ресурсов.
- `.vercelignore` содержит шаблон исключений, чтобы не заливать лишние файлы.

Если хотите, могу добавить `package.json` или настроить автоматические редиректы/реwrites — скажите, что нужно.
